Namespace: User

Built. User

users are a special class that allows adding the users functionality to your application.
features such as registration, logging in, logging out live here.
after the log in, a session cookie will be dropped, and the "authtoken" will be given in response.
This authtoken is to be supplied for subsequent requests if the the request is to be identified as being from that user
