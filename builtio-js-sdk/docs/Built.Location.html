Class: Location

Built. Location

new Built.Location(longitude, latitude)

Get new instance of Built.Location. Location class for creating a location object that can be used to set location on objects using setLocation(longitude, latitude) method.
Also, you can query objects on the basis of their geo locations where Built.Location is required to specify a location parameter.
Parameters:
Name Type Description
longitude Float longitude of the location in degrees
latitude Float latitude of the location in degrees
Returns:
returns Built.Location object instance.
Example
var myLoc= new Built.Location(longitude, latitude);  // OR 
var myLoc= new Built.Location({longitude: value, latitude: value});  //OR 
var myLoc= new Built.Location([longitude, latitude]);
