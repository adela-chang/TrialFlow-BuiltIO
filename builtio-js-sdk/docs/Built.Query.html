include(key) → {Built.Query}

include reference objects with given key in response The include parameter accepts the name of a reference field. By default, no reference field is bought along with the object, only the uids are. To include any reference, this parameter must be used. Nested references can be bought by “.” separating the references. This will work for references which are nested inside groups or references which are nested inside other references.

key String Array of reference keys to include in response.
