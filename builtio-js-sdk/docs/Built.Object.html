Class: Object

Built. Object

new Built.Object(classUid, objectAttributes)

Create, Retrieve, Update, Delete objects using Built.Object class
Parameters:
Name Type Argument Description
classUid String Class uid of the class that is created using Web UI
objectAttributes Object <optional>
 object attributes to instantiate with.
See:
  • Built.Object.extend
Examples
var myClass= Built.Object.extend("user");  
var myObject= new myClass({name:"something"});
var myObject = new Built.Object("user",{name:"something"});
